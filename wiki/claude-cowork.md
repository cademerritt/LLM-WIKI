# Claude Cowork

## What It Is
Cowork is a feature inside the **Claude desktop app** — a separate app made by Anthropic (not Claude Code).

It lets Claude:
- Sit on your computer and read your files
- Connect to your tools and apps
- Run **skills** (text-based `.md` instruction files)
- Execute tasks while you focus on other work

## How It Differs from Claude Code
- Claude Code = IDE extension (what runs in Anti-Gravity IDE)
- Cowork = standalone desktop app with a dedicated agent tab

## Skills
Skills are just `.md` files — plain text instructions Claude reads to know how to do a task. You can create them by doing a task once with Cowork, then saying "create a skill for this."

## 8 Use Cases (9x Channel Video)
1. Editing videos — cut social clips from recordings
2. Creating marketing emails — connect to tools like Customerio
3. Scraping websites — control Chrome via Claude in Chrome plugin
4. Generating brand assets — dynamic templates
5. Migrating content — Notion → Circle with images intact
6. Creating formatted reports — CSV → Google Doc
7. Triaging emails — Gmail → Airtable → Slack, runs hourly
8. Creating agent skills — turn any session into a reusable skill

## Status (April 2026)
- Available on Windows (installed)
- Not available on Linux (no official Linux release)
